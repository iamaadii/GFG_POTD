# Problem Statement
```
Given two arrays a[] and b[] of positive integers of size n and m respectively, where m ≤ n.  You are allowed to insert zeros anywhere into the second array b so that its length becomes equal to n.

The dot product of two arrays of equal length n is defined as: a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1].
Return the maximum possible dot product of the two arrays.
```

## Constraints:
```
1 ≤ m ≤ n ≤ 10^3
1 ≤ a[i], b[i] ≤ 10^3
```


# Approach-1: Recursive
- At every index i of array A, we have two choices:
  - Skip the current element A[i] and move to the next element while keeping j unchanged. 
  - Pair A[i] with the current element B[j], add their product to the answer, and move to the next indices in both arrays.

The recursive function helper(i, j) returns the maximum dot product that can be obtained starting from index i in A and index j in B.
- If both arrays are completely processed, no more contribution can be made, so return 0. 
- If array A is exhausted before B, it is impossible to pair the remaining elements of B, so return a very small value (Integer.MIN_VALUE) to mark this path as invalid.
- If array B is exhausted, no further pairs can be formed, so return 0.

For each state, we compute the maximum between:
- Skipping the current element of A.
- Taking the current pair (A[i], B[j]) and continuing with the remaining elements.

The maximum of these two choices gives the optimal answer for the current state.

---

## Code:
```java
class Solution {
	
	static int[] a;
	static int[] b;
	
	static int helper(int i, int j) {
		if (i == a.length && j == b.length)
			return 0;
		
		if (i == a.length)
			return Integer.MIN_VALUE;
		
		if (j == b.length)
			return 0;
		
		int skip = helper(i + 1, j);
		int notSkip = a[i] * b[j] + helper(i + 1, j + 1);
		
		return Math.max(skip, notSkip);
	}
	
	public int maxDotProduct(int[] A, int[] B) {
		// code here
		a = A; b = B;
		return helper(0, 0);
	}
}
```

## Complexity
```
Time Complexity: O(2ⁿ)

Let:

n = A.length
m = B.length

At each index i, you have two choices:

Skip A[i]
Pair A[i] with B[j]
```

```
Space Complexity: O(n)

The only extra space used is the recursion stack.

In every recursive call, i increases by 1.
Hence, the maximum recursion depth is n.
```

# Approach-2: Memoization
- Just store the value of current state in a dp array.

---

## Code:
```java
class Solution {
	
	static int[] a;
	static int[] b;
	static int[][] dp;
	
	static int helper(int i, int j) {
		if (i == a.length && j == b.length)
			return 0;
		
		if (i == a.length)
			return Integer.MIN_VALUE;
		
		if (j == b.length)
			return 0;
		
		if(dp[i][j]!=-1) return dp[i][j];
		
		int skip = 0 + helper(i + 1, j);
		int notSkip = a[i] * b[j] + helper(i + 1, j + 1);
		
		return dp[i][j] = Math.max(skip, notSkip);
	}
	
	public int maxDotProduct(int[] A, int[] B) {
		// code here
		a = A; b = B;
		dp=new int[a.length][b.length];
		for(int[] arr:dp){
		    Arrays.fill(arr,-1);
		}
		
		return helper(0, 0);
	}
}
```

## Complexity
```
Time = O(n*m)
```

```
Space = O(n*m) + stack-space
```
