## Problem Statement
```
Given two strings, s1 and s2, count the number of subsequences of string s1 equal to string s2.

Return the total count modulo 1e9+7.
```

### Constraints:
```
1 ≤ s1.size(), s2.size() ≤ 10^3
```


## Approach-1: Using Recursion
- Initialize two variable i=0 & j=0;
- Do a recursion:
  - Base case:
    - If 'j' reaches s2 end return 1 means we found a valid subsequence.
    - If 'i' reaches s1 return 0 means we have not found a valid subsequence.
  - If s1[i]==s2[j]: Means we found a valid character.
    - Include it & increase 'i' & 'j' both by 1.
    - Skip it & increase 'i' by 1 only.
    - Take sum of both.
  - If s1[i]!=s2[j]:
    - Increase 'i' by one only.

---

### Code:
```java
class Solution {
    static int helper(int i,int j,String s1,String s2){
        if(j==s2.length()) return 1;
        if(i==s1.length()) return 0;
        
        if(s1.charAt(i)==s2.charAt(j)){
            return ( helper(i+1,j+1,s1,s2) + helper(i+1,j,s1,s2) ) % 1000000007;
        }
        return helper(i+1,j,s1,s2);
    }
    
    public static int countWays(String s1, String s2) {
        // code here
        return helper(0,0,s1,s2);
    }
}
```

### Complexity
```
The worst case occurs when every character matches, for example:

s1 = "aaaaaa"
s2 = "aaa"

Every call branches into 2 recursive calls.

The recursion depth is at most n.
Hence,

Time Complexity=O(2^n) here n=s1.length()
```

```
Space Complexity
The maximum recursion depth is equal to the length of s1.
So,
    Space Complexity=O(n)
(due to the recursion stack)
```

## Approach-2: Memoization
---

### Code:
```java
class Solution {

    static int[][] dp;
    static int helper(int i,int j,String s1,String s2){
        if(j==s2.length()) return 1;
        if(i==s1.length()) return 0;

        if(dp[i][j]!=-1) return dp[i][j];

        if(s1.charAt(i)==s2.charAt(j)){
            return dp[i][j] = (helper(i+1,j+1,s1,s2) + helper(i+1,j,s1,s2)) % 1000000007;
        }
        return dp[i][j]=helper(i+1,j,s1,s2);
    }

    public static int countWays(String s1, String s2) {
        // code here
        dp=new int[s1.length()][s2.length()];

        for(int[] arr: dp){
            Arrays.fill(arr,-1);
        }

        return helper(0,0,s1,s2);
    }
}
```

### Complexity
```
Time = O(s1.length * s2.length)
```

```
Space = O(s1.length * s2.length) + Stack space.
```
