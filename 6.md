# Problem Statement
```
Given two integers n and k, count the number of binary strings of length n where adjacent 1 appear k times.

Since the answer can be huge, return it modulo 109+7.
```

### Constraints:
```
1 ≤ n, k ≤ 10^3
```


# Approach-1: Recursive

* Recursively build the binary string one position at a time.
* At every position, we have two choices:

    * Place `'0'`.
    * Place `'1'`.
* Maintain the following states during recursion:

    * `i` → Current position to fill.
    * `prev` → Previously placed digit (`0` for no previous digit, `1` for `'0'`, `2` for `'1'`).
    * `adjCount` → Number of adjacent `"11"` pairs formed so far.
* If we place `'0'`, the adjacent pair count remains unchanged.
* If we place `'1'` and the previous digit was also `'1'`, increment `adjCount` by `1`.
* If `adjCount` becomes greater than `k`, stop exploring that path since it can never produce a valid answer.
* Once all `n` positions have been filled (`i == n`):

    * If `adjCount == k`, we have found a valid binary string, so return `1`.
    * Otherwise, return `0`.
* The final answer is the sum of both recursive calls at every position, taken modulo `10^9 + 7`.


---

### Code:
```java
class Solution {
    
    static int n,k;
    
    static int helper(int i,int prev,int adjCount){
        if(adjCount>k){
            return 0;
        }
        if(i==n){
            if(adjCount==k){
                return 1;
            }
            return 0;
        }
        
        int a = helper(i+1,1,adjCount);
        int b = helper(i+1, 2, adjCount + (prev==2 ? 1 : 0));
        
        return (a + b) % 1000000007;
    }
    
    public int countStrings(int N, int K) {
        // code here
        n=N; k=K;
        return helper(0,0,0);
    }
}
```

### Complexity
```
Time = O(2^n) As we are making two function call for each postion
```

```
Space = O(n) recursion stack space.
```

# Approach-2: Memoization
- Just create a 3d array of size (n+1) * (3) * (k+1) in-order to prevent from recalculation.
- At every return statement add value in current index, prev, countAdj cell.

---

### Code:
```java
class Solution {
    
    static int n,k;
    static int[][][] dp;
    
    static int helper(int i,int prev,int adjCount){
        if(adjCount>k){
            return 0;
        }
        if(i==n){
            if(adjCount==k){
                return 1;
            }
            return 0;
        }
        
        if(dp[i][prev][adjCount]!=-1) return dp[i][prev][adjCount];
        
        int a = helper(i+1,1,adjCount);
        int b = helper(i+1, 2, adjCount + (prev==2 ? 1 : 0));
        
        return dp[i][prev][adjCount] = (a + b) % 1000000007;
    }
    
    public int countStrings(int N, int K) {
        // code here
        n=N; k=K;
        dp = new int[n+1][3][k+1];
        
        for(int i=0;i<n+1;i++){
            for(int j=0;j<3;j++){
                for(int m=0;m<n;m++){
                    dp[i][j][m] = -1;
                }
            }
        }
        
        return helper(0,0,0);
    }
}
```

### Complexity
```
Time = O(n * 3 * k)
```

```
Space = O(n * 3 * k) + stack space
```

# Approach-3: Tabulation

---

### Code:
```java
class Solution {
	
	static int n, k;
	static int[][][] dp;
	
	static int helper(int i, int prev, int adjCount) {
		if (adjCount>k) {
			return 0;
		}
		if (i == n) {
			if (adjCount == k) {
				return 1;
			}
			return 0;
		}
		
		if (dp[i][prev][adjCount] != -1)
			return dp[i][prev][adjCount];
		
		int a = helper(i + 1, 1, adjCount);
		int b = helper(i + 1, 2, adjCount + (prev == 2 ? 1 : 0));
		
		return dp[i][prev][adjCount] = (a + b) % 1000000007;
	}
	
	public int countStrings(int N, int K) {
		// code here
		n = N; k = K;
		dp = new int[n + 1][3][k + 2];
		dp[n][1][k] = 1;
		dp[n][2][k] = 1;
		
		for (int i = n - 1; i >= 0; i--) {
			for (int prev = 0; prev <= 2; prev++) {
				for (int adjCount = 0; adjCount <= k; adjCount++) {
					int a = dp[i + 1][1][adjCount];
					int b = dp[i + 1][2][adjCount + (prev == 2 ? 1 : 0)];
					dp[i][prev][adjCount] = (a + b) % 1000000007;
				}
			}
		}
		return dp[0][0][0];
	}
}
```

### Complexity
```
Time = O(n * 3 * k);
```

```
Space = O(n * 3 * k)
```
