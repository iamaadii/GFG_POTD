## Problem Statement
```
Given a floor of dimensions n × m and an unlimited supply of tiles of size 1 × m, find the total number of ways to completely tile the floor.
Each tile can be placed in one of the following ways:

Horizontally, covering 1 row and m columns.
Vertically, covering m rows and 1 column.

Since the number of possible tilings can be very large, return the answer modulo 109+7.

Note: n and m are positive integers, and m ≥ 2.
```

### Constraints:
```
1 ≤ n ≤ 10^5
2 ≤ m ≤ 10^5
```


## Approach-1: Recursive
- To count the number of ways to tile an n × m floor using 1 × m tiles, we consider the placement of the first tile.
- Place the tile horizontally: A horizontal tile covers one complete row, so we are left with row - 1 rows to tile. This contributes helper(row - 1, col) ways. 
- Place the tiles vertically: To cover the leftmost m × 1 section, we need to place m vertical tiles together. This is possible only if row >= col. After placing them, the remaining floor has row - col rows, contributing helper(row - col, col) ways. 
- Since these two choices are independent, the total number of ways is the sum of both possibilities. 
- The recursion terminates when row == 0, which represents a completely tiled floor and counts as one valid arrangement.

---

### Code:
```java
class Solution {
    
    static int helper(int row,int col){
        if(row==0){
            return 1;
        }
        
        int ans = 0;
        ans = (ans + helper(row-1,col)) % 1000000007;
        if(row>=col){
            ans = (ans + helper(row-col,col)) % 1000000007;
        }
        return ans;
    }
    
    public int countWays(int n, int m) {
        // code here
        
        return helper(n,m);
    }
}
```

### Complexity
```
Time = O(2^n)
```

```
Space = O(n) Stack space
```

## Approach-2: Memoization


---

### Code:
```java
class Solution {
    
    static int[] dp;
    static int helper(int row,int col){
        if(row==0){
            return 1;
        }
        
        if(dp[row]!=-1) return dp[row];
        
        int ans = 0;
        ans = (ans + helper(row-1,col)) % 1000000007;
        if(row>=col){
            ans = (ans + helper(row-col,col)) % 1000000007;
        }
        return dp[row]=ans;
    }
    
    public int countWays(int n, int m) {
        dp=new int[n+1];
        Arrays.fill(dp,-1);
        
        return helper(n,m);
    }
}
```

### Complexity
```
Time = O(n)
```

```
Space = O(n) + stack space
```

## Approach-3: Tabulation


---

### Code:
```java
class Solution {
    public int countWays(int n, int m) {
        int[] dp=new int[n+1];
        dp[0] = 1;
        
        for(int row=1;row<=n;row++){
            int ans = dp[row-1];
            if(row >= m){
                ans = (ans + dp[row-m]) % 1000000007;
            }
            dp[row]=ans;
        }
        
        return dp[n];
    }
}
```

### Complexity
```
Time = O(rows)
```

```
Space = O(rows)
```
