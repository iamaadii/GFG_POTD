## Problem Statement
There are infinitely many people standing in a row, indexed from 1. The strength of the person at index i is i².

Given a strength p, determine the maximum number of people that can be defeated. A person with strength x can be defeated only if p ≥ x, after which the strength p decreases by x.

### Constraints:
1 ≤ p ≤ 3*10^8

```
Input: p = 14
Output: 3
Explanation: The strengths of the first few people are 1, 4, 9, 16, .... Defeating the first three people consumes 1 + 4 + 9 = 14 strength, leaving 0. Therefore, the maximum number of people that can be defeated is 3.
```

## Approach-1:
- Initialize variable 'i' with 1 & 'count' with 0.
- Run a while loop until p>=i*i & inside the loop 
  - increase 'count' by 1.
  - decrease 'p' by i*i.
  - increase 'i' by 1.
- Return count.

---

### Code:
```java
class Solution {
    int maxPeopleDefeated(int p) {
        // code here
        int i=1;
        int count=0;
        
        while(p>=i*i){
            count+=1;
            p-=i*i;
            i+=1;
        }
        return count;
    }
};
```

### Complexity
```dtd
The loop runs while:

i² ≤ remaining p

The number of iterations is approximately the largest k such that:

1² + 2² + 3² + ... + k² ≤ p

Using the formula:

1² + 2² + ... + k² = k(k + 1)(2k + 1) / 6 ≈ k³ / 3

So:

k³ ≈ p

k ≈ ∛p
```
- Time : O(∛p)
- Space : O(1)

## Approach-2: Using binary Search

---

### Code:
```java
class Solution {
    int maxPeopleDefeated(int p) {
        long low = 0, high = 2000; // sufficient for constraints

        while (low <= high) {
            long mid = low + (high - low) / 2;

            long sum = mid * (mid + 1) * (2 * mid + 1) / 6;

            if (sum <= p)
                low = mid + 1;
            else
                high = mid - 1;
        }

        return (int) high;
    }
}
```

### Complexity
```dtd
Binary search runs in O(log k)
        Since k ≈ ∛p
       O(log∛p))=O(logp)
```
- Time : O(log p)
- Space : O(1)
