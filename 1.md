## Problem Statement
Given two integers a and b in the form of strings. Return the last digit of ab.

### Constraints:
- 1 ≤ a.size(), b.size() ≤ 1000
- a and b consist only of numeric digits ('0' - '9')
- a and b do not contain any leading zeros, except when number itself is "0"

---
# Approaches

---

## Optimal:
### 1. Handle the Base Case
- If `b = 0`, then:

  \[
  a^0 = 1
  \]

- Therefore, return `1`.

### 2. Observe the Pattern of Last Digits
- The last digit of powers of a number repeats in cycles.

#### Examples
- `2 → 2, 4, 8, 6` (cycle length = 4)
- `3 → 3, 9, 7, 1` (cycle length = 4)
- `4 → 4, 6` (cycle length = 2)

- Store these cycles for digits `0` to `9`.

### 3. Find the Relevant Base Digit
- Only the last digit of `a` affects the last digit of:

  \[
  a^b
  \]

- Extract the last digit of `a`.

### 4. Get the Cycle Length
- From the stored cycle corresponding to the base digit, determine its length `len`.

### 5. Compute `b mod len`
- Since `b` can be extremely large, converting it directly into an integer is not feasible.
- Traverse the digits of `b` and continuously compute:

  \[
  rem = b \bmod len
  \]

- This remainder determines the position within the cycle.

### 6. Handle the Case When Remainder is 0
- If `rem = 0`, it means we need the **last element** of the cycle.
- Therefore, set:

  \[
  rem = len
  \]

### 7. Return the Answer
- The required last digit is the element at index:

  \[
  rem - 1
  \]

  in the corresponding cycle.

---

### Code:
```java
class Solution {
    public int getLastDigit(String a, String b) {
        // code here
        
        if(b.equals("0")) return 1;
        
        List<int[]> l = new ArrayList<>();
        l.add(new int[]{0});
        l.add(new int[]{1});
        l.add(new int[]{2,4,8,6});
        l.add(new int[]{3,9,7,1});
        l.add(new int[]{4,6});
        l.add(new int[]{5});
        l.add(new int[]{6});
        l.add(new int[]{7,9,3,1});
        l.add(new int[]{8,4,2,6}); 
        l.add(new int[]{9,1}); 
        
        int base = a.charAt(a.length()-1)-'0';
        int len = l.get(base).length;
        
        int rem = 0;
        for(char c:b.toCharArray()){
            rem = (rem*10 + (c-'0'))%len;
        }
        
        if(rem==0){
            rem=len;
        }
        
        return l.get(base)[rem-1];
    }
}
```
